# tcp-port-scanner
🔍 Multi-Threaded TCP Port Scanner
A fast, lightweight, and efficient TCP port scanner written in Python. This tool allows users to scan a target host for open ports within a specified range using multi-threading for speed.

⚡ Features
✅ Multi-threaded scanning for high speed
✅ Custom port range input
✅ Error handling for better stability
✅ Lightweight & easy to use
✅ No external dependencies – runs with built-in Python modules
